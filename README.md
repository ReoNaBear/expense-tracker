# Expense-tracker

## 功能

1.使用者可以創建帳號密碼

2.使用者可以登入登出

3.使用者可以用FB登入

4.使用者可以創建一筆資料 (新增支出button)

5.使用者可以修改一筆資料 (修改button)

6.使用者可以刪除一筆資料 (刪除button)

7.使用者可以依種類或是時間查詢資料

8.首頁會列出所有的支出項目


## 使用方法


### 部署步驟
-------------

- Git clone this project.
```
https://github.com/ReoNaBear/expense-tracker.git
```

- Npm 
```
npm install
```

- Use seeder to create data
```
npm run seed
```

- Start
```
npm run dev
```
